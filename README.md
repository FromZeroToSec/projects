# FromZeroToSec — Portfolio DevSecOps

Portfolio progressif de projets Python orientés sécurité (authentification, chiffrement, scan réseau, audit, automatisation).
Objectif : passer d'un parcours hôtellerie à une carrière DevSecOps d'ici 2030, un projet à la fois.

## 🗺️ Structure

Ce repo regroupe les projets par bloc de compétences croissant :

| Bloc | Thème | Projets |
|------|-------|---------|
| 1 | Python Fondamentaux | 01–10 |
| 2 | Linux & Réseau | 11–20 |
| 3 | Sécurité Défensive | 21–35 |
| 4 | DevSecOps | 36–50 |
| 5 | Projets Avancés | 51–70 |

## 📁 Organisation d'un projet

Chaque dossier contient :
- `main.py` (ou plusieurs fichiers `.py` si nécessaire)
- `README.md` propre au projet (description, usage, ce qu'il démontre techniquement)
- `.gitignore` si le projet a des fichiers sensibles ou générés (venv, secrets, cache)

## 🛠️ Stack

- Python 3
- Git / GitHub CLI (`gh`)
- Modules : `hashlib`, `json`, `os`, `re`, `cryptography`, `pathlib`, `random`, `sys`
