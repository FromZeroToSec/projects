# 01 - Calculatrice

Premier projet Python — calculatrice basique avec inputs utilisateur.

## Ce que ça fait
- Demande deux nombres à l'utilisateur
- Demande l'opération souhaitée (+, -, ÷, x)
- Effectue le calcul et affiche le résultat
- Gère la division par zéro
- Tourne en boucle jusqu'à ce que l'utilisateur quitte

## Usage
```bash
python main.py
```

## Exemple
Enter the first number: 5
Enter your operation (+,-,÷,x): +
Enter the second number: 3
Result: 5.0 + 3.0 = 8.0
Again? (y/n): n
